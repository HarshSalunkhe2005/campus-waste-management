# Campus Waste Management System

A Flask and MySQL web application for managing campus food donations.

---

## Prerequisites
* You must have **Python 3** and **MySQL Server** installed on your computer.

---

## How to Run
1.  Download the project as a ZIP and unzip it to a folder on your computer.
2.  Make sure your MySQL server is running.
3.  Double-click the **`run_setup.bat`** file.(Please check batch file contents and update the paths accordingly!!)
4.  Follow the on-screen prompts (you will need to enter your MySQL `root` password).

The script will automatically set up everything else and open the application in your browser.
